# ⚡ e-spire-mobile

## 📖 Sobre 
O **e-spire-mobile** é o aplicativo oficial do ecossistema e-spire, construído com o framework Flutter. O principal objetivo da plataforma é conectar pessoas através de atividades físicas e hobbies em comum, como yoga ou corrida. A aplicação mobile foca na experiência do usuário e na construção de comunidades, integrando mapas para visualização de eventos, um feed social interativo e um sistema de recomendações de novos grupos.

Por ser desenvolvido em Flutter e Dart, o projeto possui suporte multiplataforma, abrangendo sistemas mobile (Android e iOS), além de Web e Desktop (Windows, macOS, Linux).

## 🛠️ Configuração do Projeto (Setup)

### Pré-requisitos
Certifique-se de ter o **Flutter SDK** instalado e devidamente configurado em sua máquina, além dos emuladores de sua preferência (Android Studio, Xcode, etc).

### Instalação

1. Clone o repositório:
```bash
git clone [https://github.com/YvesDeSa/e-spire-mobile.git](https://github.com/YvesDeSa/e-spire-mobile.git)
cd e-spire-mobile

```

2. Instale as dependências:

```bash
flutter pub get

```

### Executando a aplicação

```bash
# Rodar no dispositivo/emulador conectado
flutter run

# Rodar especificando uma plataforma (exemplo: Chrome)
flutter run -d chrome

```
